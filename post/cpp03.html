C++变量

什么是变量

首先，抛出一个问题，什么是变量？
按字面意思来讲，就是值可以改变的量。一个变量应该有一个名字，并在内存中占据一定的内存单元，在该存储单元中存放变量的值。

注意区分 变量名 和 变量值 这两个不同的概念。

基本的变量类型

类型 描述
int 对机器而言，整数的最自然的大小。
float 单精度浮点值。单精度是这样的格式，1位符号，8位指数，23位小数。
double 双精度浮点值。双精度是1位符号，11位指数，52位小数。
bool 存储值 true 或 false。
char 通常是一个字符（八位）。这是一个整数类型。
void 表示类型的缺失。即无类型。

C++ 也允许定义各种其他类型的变量，比如枚举、指针、数组、引用、数据结构、类等等，这将会在后续的章节中进行讲解。

变量名规则

变量名是标识符的一种。C++规定标识符只能由字母、数字和下划线3种字符组成，且第一个字符必须为字母或下划线。
sum _sum 是合法的，而123sum是非法的。

注意：在C++中，大写字母和小写字母被认为是两种不同的字符，所以sum和SUM是两个不同的变量名。

定义变量的其他问题

在C++中，未经定义的变量是不允许使用的，在使用任意一个变量之前都要进行定义，否则就会报错。
定义后的变量的类型会被指定，在编译时就能为其分配存储单元，且进行运算时也便于检查运算是否合法。
例如整型变量a和b，可以进行求余运算 a%b
但如果变量a和b的类型为浮点数，则无法进行取余运算。

对变量赋初值

在定义变量时可以对它赋予一个初值，称为变量初始化。初值可以是常量，也可以是一个有确定值的表达式。
如float a,b = 1.23,c = 2*3中变量a未进行初始化，变量b初值为1.23，变量c的初值为2*3即6。
注意：如果未对变量进行初始化，则其初值是一个不可预测的值。初始化不是在编译阶段完成的，而是在程序运行本函数时赋予初值的，相当于一个赋值语句
