主函数部分

一个函数由两部分组成

  1. 函数首部
  2. 函数体

int main() 为主函数首部，即定义整型函数 main，其中 int 为函数类型为整型，main 为函数名，() 一对圆括号中为函数属性。

一个函数名后面必须跟一对圆括号，函数参数可以省略，例如int main()

函数体即为 { } 这对花括号括起来的内容，{ }既可以在函数中使用，也可在 if 判断语句、for 循环语句中使用，用来界定范围，写起来比较规范。
