基本的内置数据类型

计算机处理的对象是数据，而数据又分为多种类型。
C++的数据包括常量和变量，他们都具有类型。

基本类型 关键字
整型 int
浮点型 float
双浮点型 double
字符串型 char
布尔型 bool
无类型 void

一个基本类型可以是使用一个或多个类型修饰符进行修饰：

  • signed
  • unsigned
  • short
  • long

这里涉及到各种变量类型在内存中存储需要占用的内存，如果看完这里没有理解可以暂时跳过，等到有一定基础时再回到这里查看

整型数据

整型数据分为长整型（long int）、一般整型（int）、短整型（short int）。在 int 前加 long 和 short 分别表示长整型和短整型。

C++中没有规定每一种数据所占的内存大小，只规定 int 型数据所占的字节数不大于 long 型，不小于 short 型。

ps: 不同系统会有所差异，一字节为 8 位。
ps: 默认情况下，int、short、long都是带符号的，即 signed。
ps: long int 8 个字节，int 都是 4 个字节，早期的 C 编译器定义了 long int 占用 4 个字节，int 占用 2 个字节，新版的 C/C++ 标准兼容了早期的这一设定。

下面贴上一个有用但不完全有用的表格

类型 位 范围
int 4个字节 -2147483648 到 2147483647
unsigned int 4个字节 0 到 4294967295
signed int 4个字节 -2147483648 到 2147483647
short int 2个字节 -32768 到 32767
unsigned short int 2个字节 0 到 65,535
signed short int 2个字节 -32768 到 32767
long int 8个字节 -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807
signed long int 8个字节 -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807
unsigned long int 8个字节 0 到 18,446,744,073,709,551,615

值得一提的是：变量的大小会根据编译器和所使用电脑的操作系统而有所不同。

浮点型数据

浮点型（又称实型）数据分为单精度（float）、双精度（double）和长双精度（long double）3种。

类型 位 范围
float 4 个字节 精度型占4个字节（32位）内存空间，+/- 3.4e +/- 38 (~7 个数字)
double 8 个字节 双精度型占8 个字节（64位）内存空间，+/- 1.7e +/- 308 (~15 个数字)
long double 16 个字节 长双精度型 16 个字节（128位）内存空间，可提供18-19位有效数字。

修饰符 unsigned 和 signed

在整型符号 int 和 字符型符号 char 的前面，可以加修饰符 signed（“有符号”）和 unsigned（“无符号”）。

  • 如果指定为 signed ，则数据以补码形式存放，存储单元中的最高位（bit）用来表示数值的符号。
  • 如果指定为 unsigned ，则数值没有符号，全部二进制位都用来表示数值本身。

有符号时，能存储的最大值为2^15-1，即32767，最小值为-32768。无符号时，能存储的最大值为2^16-1，即65535，最小值为0。有些数据是没有负值的（如学号、货号、身份证号），可以使用 unsigned ，他存储正数的范围比用 signed 时扩大一倍。
